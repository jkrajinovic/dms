## Development environment

Install docker [https://www.docker.com/](https://www.docker.com/)

In command prompt run:

```bash
docker-compose up svelte
```

On linux machine go to:
http://0.0.0.0:5000

On windows machine go to:
http://localhost:5000


Happy coding.
